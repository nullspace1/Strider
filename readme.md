# Strider

Strider is an application designed to help running coaches prepare plans for their athletes, and to organize their day-to-day training sessions.

# Features

## Athelete Information

Usign Strider, coaches can keep track of their athletes' characteristics and stats, such as:

- Age
- Weight
- Height
- Paces (VO2max, 80%, etc...)

These values are easily be viewed, edited and tracked over time.

## Plans 

Training plans are split into mesocycles each of which split into week-long microcycles. Using the information provided for each athlete, the application will provide assistance while creating individual sessions. The tool will automatically calculate different effort levels and provide a series of templates. Additionally, the user will be able to store templates to quickly create training sessions.
